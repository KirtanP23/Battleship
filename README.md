# Battleship Game

## Overview
This project is a 1-player version of the classic Battleship game built for COSC 2100 - Assignment 2.  
The computer randomly places five ships on a hidden game board, and the player must try to sink them all in as few turns as possible.

## Features
- 10x10 game board
- Random ship placement
- Mouse-based firing by clicking board squares
- Keyboard-accessible coordinate selection
- Hit and miss tracking
- Turn counter
- Boat progress tracking
- Win notification when all ships are sunk
- Reset option to start a new game

## Ships Included
The game includes the following ships:
- Carrier (5 spaces)
- Battleship (4 spaces)
- Cruiser (3 spaces)
- Submarine (3 spaces)
- Destroyer (2 spaces)

## How to Play
1. Start the game.
2. Choose a target square on the board.
3. Fire a missile by clicking a square or using keyboard controls.
4. The board will display whether your shot was a hit or a miss.
5. Continue until all ships are sunk.
6. Try to finish the game in the lowest number of turns possible.
